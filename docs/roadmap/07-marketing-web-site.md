# Шаг 07. Marketing Web Site

## Цель

Создать публичный маркетинговый сайт `web` на 5-7 страниц с сильной SEO-подготовкой.

## Объем работ

### 1. Определить стек

Так как сайт SEO-heavy, нужно использовать web stack, ориентированный на:
- SSR или SSG
- удобную SEO-настройку
- хорошую индексацию поисковиками

### 2. Реализовать структуру страниц

Минимальный состав:
- Home
- Features / How it works
- Screenshots
- Download
- Privacy Policy
- Terms / Legal
- Contact или FAQ

### 3. Подготовить контентную структуру

Нужно предусмотреть:
- hero section
- product messaging
- ссылки на загрузку приложений
- скриншоты приложения
- описание Morning/Night сценариев

### 4. Реализовать SEO-конфигурацию

Нужно заложить:
- title/meta description на каждую страницу
- canonical URLs
- sitemap
- robots.txt
- Open Graph
- Twitter cards
- structured data
- image optimization

### 5. Подготовить analytics/search integrations

Архитектура должна позволять позже подключить:
- Google Search Console
- analytics
- verification tags поисковиков

## Ограничения

- `web` не должен тянуть серверные секреты
- если используются API вызовы, они должны быть только публичными и безопасными
- юридические страницы должны быть доступны по стабильным URL

## Результат шага

Появляется готовый к индексации публичный сайт продукта с сильной SEO-базой.

## Критерии приемки

- сайт имеет 5-7 страниц
- все ключевые страницы индексируемы
- sitemap и robots работают
- для страниц настроены meta/OG/canonical
- ссылки на скачивание приложения и legal pages доступны
